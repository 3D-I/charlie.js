Charlie.js

Sync CSS animations with video

Tweet

HTML5 video and CSS3 animations — Charlie.js is what happens when you mix the two together.

Synchronize animations to specific times in an embedded video with an easy to use API. If the video pauses, the animations pause.

See the Mac Pro marketing page and this flexbox tutorial for examples of what you can do when you combine animations with video.

1. Attach to a video

CHARLIE.setup(video);

2. Create some animations

.grow {
    animation-name: grow;
    animation-duration: 4s;
    animation-fill-mode: forwards;
}

@keyframes grow {
   ...
}

3. Link it to your HTML with data attributes

<div class="charlie" data-animations="grow, shrink" data-times="40.2, 90.3">
</div>
